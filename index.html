◈ CONTENT 01 — 비동기 PVP

카오스 페스티벌

상대 편성을 클론으로 불러와 클라이언트에서 승패를 판정하는 비동기 PvP — 설계부터 라이브까지 E2E 담당
  • ◢공정성 — 서버 발급 시드 기반 결정적 시뮬레이션. 어떤 기기에서도 같은 매치는 같은 결과.
  • ◢안정성 — 두 전장의 간섭을 진영 격리로 차단하고, 위반 코드를 빌드 타임에 잡는 정적 검증기를 직접 제작.
  • ◢수익 루프 — 칩 가챠(50연 천장·확률 공시), 15티어 랭킹·주간 시즌, 온보딩 튜토리얼 구현.

  • Live/Record Clone 매칭 — 실유저 스냅샷(LIVE)과 기록 고스트(RECORD)를 클론 스폰해 3라운드(2선승) 대전. 자기 클론과 매칭되면 닉네임 위장·편성 셔플로 노출 방지.
  • 결정론 구현 — 서버 발급 match_seed로 전용 RNG를 격리하고, 런타임별 편차가 없는 FNV-1a 해시로 시즌 카드 선정까지 전 클라이언트 동일 결과 보장.
  • 진영 격리 — 상대 전장을 월드 오프셋으로 분리하고, 진영별 버킷 기반 무할당(alloc-free) 타게팅 헬퍼로 교차 간섭 차단.
  • 정적 검증기 — 데미지 계산 호출의 진영 인자 누락을 소스 스캔으로 검출. 에디터 메뉴와 CI 종료코드 연동으로 격리 위반이 빌드를 통과하지 못하게 설계.
  • 서버 연동 10종 — 시즌 진입/리셋, 편성 저장, 매칭·결과 제출, 랭킹(상위 500+내 순위), 듀얼 4종. 서버 프로그래머와 DTO 스펙 협의.
◈ CONTENT 02 — 시즌제 경쟁 던전

방주 (드랍쉽)

웨이브마다 모듈 가챠로 빌드를 완성하며 시즌 랭킹을 겨루는 던전 — 12개 서비스 위임 구조로 설계
  • ◢아키텍처 — 매니저는 상태·저장만, 가챠/상점/시즌/순찰 등 12개 도메인 서비스로 로직 분리. 이후 카오스 페스티벌의 레퍼런스가 됨.
  • ◢가챠 신뢰성 — 등급 소진을 사전 체크하고 전 등급 소진 시 폴백 경로로 뽑기 실패를 원천 차단.
  • ◢기획 협업 툴 — 확률·등급 테이블을 기획이 직접 검증하는 가챠 시뮬레이터 에디터 툴 자체 제작.
