라이브 서비스 중인 모바일 타워디펜스 게임. 비동기 PvP 카오스 페스티벌과 시즌제 경쟁 던전 방주(드랍쉽) — 두 대형 콘텐츠를 서버 스펙 협의부터 라이브 운영까지 수직 슬라이스 전체로 책임지고, 수익화 이벤트·리텐션 시스템까지 시스템 단위로 담당했습니다.

◢ CONTENT 01 — 카오스 페스티벌 · 비동기 PvP
  • ◢Live/Record Clone 매칭 — 실유저 스냅샷(LIVE)과 기록 고스트(RECORD)를 클론 스폰해 3라운드(2선승) 대전. 자기 클론과 매칭 시 닉네임 위장·편성 셔플로 노출 방지.
  • ◢결정적 시뮬레이션 — 서버 발급 match_seed + 환경 독립 해시(FNV-1a)로 어느 클라이언트에서나 동일한 결과를 보장해 PvP 공정성 확보.
  • ◢진영 격리 + 자체 정적 검증기 — 월드 오프셋 분리와 진영별 버킷 무할당 타게팅으로 양 필드 간섭 차단. 데미지 격리 누락을 소스 스캔으로 검출하는 검증기를 직접 제작해 에디터 + CI 종료코드로 빌드 타임 차단.
  • ◢서버 연동 10종 — 시즌/편성/매칭/랭킹/듀얼 엔드포인트. 15티어 랭킹, 주간 시즌, 칩 가챠(50연 천장 + 확률 공시), 3단계 온보딩 튜토리얼 구현.
